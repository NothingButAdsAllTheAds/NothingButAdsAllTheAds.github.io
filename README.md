# Documentation Stuffs

## Style Guide

### Color Schemes

The approved colors are:

- Red (#FF0000)
- Light Gray (#D3D3D3)
- White: (#FFFFFF)
- Black: (#000000)

Approved Fonts
- [Roboto Condensed](https://fonts.google.com/specimen/Roboto+Condensed) (for titles)
- [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) (for main text)
